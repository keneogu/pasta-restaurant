## Pasta-Restaurant

> This mini-app is made with React and its purpose is to understand the concept of react.


## Built With 

- JavaScript, React, HTML5 and CSS

### Prerequisites

- Git
- NodeJs
- any Modern Browser

## Getting Started

To get a local copy up and running follow these simple example steps.

- git clone `git@github.com:keneogu/pasta-restaurant.git`
- cd into the `development`
- run `npm install`
- run `npm run start`
